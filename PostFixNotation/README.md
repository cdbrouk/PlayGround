## Convert infix into post fix

1. 비연산자는 결과 문자열로 바로 저장
2. 연산자는 스택으로
3. 식의 연산자가 스택의 연산자의 우선순위보다 낮거나 같은경우(<=) 스택연산자보다 높아질 때까지 pop
4. 높은 경우는 무조건 push
5. "("도 무조건 push
6. ")"은 "(" 만날때까지 pop
